Quantified Repitition

metacharacter
meaning
{
starting quantified repitition
}
ending quantified repitition

as apposed to metacharacter, this allows us to specify exactly how many times we want character repeated

{min,max}

this means the {minimum times it can be repeated, maximum times it can be repeated}

  • min always must be include, and can be zero

  • maximum value is optional

Three syntaxes

  • \d{4,8}

    matches number between 4 and 8 digits

  • \d{4}

    matches numbers 4 digits (maximum is assumed to be the minimum)

  • \d{4,}

    matches numbers 4 or more digits (max is infinite)

    \d{4,} would be the same as \d*

    \d{1,} would be the same as \d+ why?

Good Example using this method

  • Phone numbers
    • \d{3}-\d{3}-\d{4}
  • Finaincials documents like bonds
    • A{1,2} would match "A" Bonds and "AA" Bonds but NOT "AAA Bonds"

the \s will limit the search to one line because remember, \s is white space so (new line, line return, space, or tab)
