Line Breaks and Multiline Mode

Metatcharacter Meaning
^ Start of string/line
$ End of string/line
\A Start of string, never end of line
\Z End of string, never end of line

this does not match any of the subsequent line, why?

because there are two different modes

  • single line
    • is default, so ^ and $ do not match line break
    • as well as \A and \Z (Remeber these two do not work in javascript)
  • multiline mode
    • ^ and $ will work
    • \A and \Z won't work

most unix regex tool will only support single line mode
multiline came later
