WildCard .

The wild card character . (period) matches anything except new line, or line return

/h.t/ matches "hat", "hot", "hit" but not "heat"

because the wildcard character only matches one single character, including a space or special character

/9.00/ will match "9.00" "9300" or "9-00"

but be careful and rememember, a period is not a period, it is a wild card

we have to us an escape character to use a period as an period


Escape Character \

The escape character \ (backslash) allows us to use a metacharacter "literally" in a regular expression

it treats the regex character as a literal character

Match a period with /\./

/9\.00 matches "9.00", but not "9500" or "9-00"

You can escape a \ backslash with another backslash /\\/

Another important note to remember, quotes are NOT metacharacters and do not need to be escaped


In the following example, the period works as a wild card because the period is a character

but to be safe, it is good practice to escape the period at the end

another example, gives us just the period and not the wildcard underscore _

Other Special Characters

  • - Spaces " "
  • - Tabs \t
  • - Line Returns
    • (\r) line return
    • (\n) new line
    • (\r,\n, \r\n)

    • (\n\r) both line return and new line

To match a Space, we need to type a literal space " "

to match a Tab we need to use \t

These are refered to as control character
