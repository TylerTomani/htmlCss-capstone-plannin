Character Ranges

Range character -

the - ( dash or hypen) is the range character

Instead of typing out

  • /abcdefghijklmnopqrstuvwxyz/
  • /ABCDEFGHIJKLMNOPQRSTUVWXYZ/
  • /0123456789/

we can use

  • /a-z/
  • /A-Z/
  • /[0-9]/

the - (dash or hyphen) is only a metacharacter within a character set, otherwise it is a literal character


Caution: common mistake

/[50-99]/ will not match numbers between 50 and 99 because it can only match one character or digit

this would match the character/digit 0 or 5 to the number of nine so 0-9

It is NOT a number range but a character range
