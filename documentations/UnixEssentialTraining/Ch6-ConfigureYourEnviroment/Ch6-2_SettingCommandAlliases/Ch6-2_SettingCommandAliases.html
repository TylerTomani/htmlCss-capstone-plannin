Setting Command Alliases

to get currently defined aliases type:
alias

this is telling us that cp is the same as 'cp -i', df is 'df -h

we can make a command by typing alias followed by what we want it to equate to

                ll ='ls -lah'

now, whenever we type ll is will actually be the ls -lah command

but, when we type an alias like this it doesn't stick aroudn
we need to make changes in the ~/.zshrc file

                nano ~/.zshrc

whenever we make changes to the file we need to open a new terminal window or
type the command source followed by shell folder

                source ~/.zshrc

if you find yourself always typing a command with certain options
make it an alias!
