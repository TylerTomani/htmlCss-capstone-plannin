Setting Permissions

Alpha Notation

user group other
Read r r r
Write w w -
Execute x - -
rwx rw x

Changing Permissions

chmod "change mode"

Example:

  • chmod ugo=rwx filename
  • chmod u=rwx,g=rw,o=x filename

You can use "+" or "-" to add or take away permissions

Octal Notation

user group other
Read 4 4 4
Write 2 2 0
Execute 1 0 0
7 6 4

Important Reminder!

to change directory permissions use chmod 741 directoryName

BUT

this will only change the directory to u=rwx,g=rw,o=x and NOT the files inside

Remember to use Recursice -R

chmod -R 777 somedirectory

will change direcotry and its files to ugo=rwx
