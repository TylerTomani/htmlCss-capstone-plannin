File Directory and Ownership

Owner

the owner appears in the 3rd column

Group

the group is in the 4th column

the parent directory has a different owner and group

Change Ownership

chown user:group somefile.txt

will change user and owner

chown user somefile.txt

will just change owner

chown :group somefile.txt

will only change group


changing the owner of a directory will not change the ownership of the files inside

to do that we need to use the recursive option

chown -R user somedirectory

Change owner not permitted

in order to change the owner of a file you must be the root user or use a command called

sudo
