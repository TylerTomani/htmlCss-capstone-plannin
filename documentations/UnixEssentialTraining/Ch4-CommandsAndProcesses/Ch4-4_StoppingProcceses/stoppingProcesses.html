Kill proccess

we can kill a process using the "PID" number found in top which stands for "process I.D"

Try it with terminal window

  • open a new terminal window
  • In one of the terminal windows, type the command ps
  • type in kill command followed by the PID number of the new terminal window

    • kill 16860

  • the command kill alone usually works but sometimes doesn't and won't with terminal,
  • when it doesn't work us the -9 option
