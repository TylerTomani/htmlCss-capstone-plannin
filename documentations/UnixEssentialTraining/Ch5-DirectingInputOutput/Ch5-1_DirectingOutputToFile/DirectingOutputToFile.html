Directing Output to a File

Standard Input and Output

  • Standard Input: stdin
    • unix using keyboard as stdin by default
  • Standard Output: stdout
    • default is text terminal

Direct output to a file

command > file

on the right side of the > we put a path to a file where the output of the command is directed

if no path is specified it get saved to the current working direcotry

if we direct output to a file containg something it will Overwrite that content

so be careful

combining files with cat

we can use cat to combine to files and send it to a new file

cat people.txt hello.txt > cat_file.txt

appending files

instead of erasing content of a file we can append contents of a file to another using

command >> file
